# TaskFlow Full Working Project

This is a full working single-file Project Management Tool.

## How to Run

1. Extract the ZIP file.
2. Open `index.html` in Chrome.
3. Click `Login Dashboard`.
4. Use demo login.
5. Create tasks, add comments, and move tasks:
   - To Do
   - In Progress
   - Completed

## Features

- Modern UI
- Login / Logout
- Create Task
- Add Comments
- Move Task Status
- Project Stats
- Responsive Layout
