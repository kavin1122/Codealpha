# CodeAlpha_EcommerceStore

Modern animated e-commerce store project for CodeAlpha Full Stack Development Internship.

## Features

- Modern landing page
- Animated product cards
- Product search
- Category filter
- Wishlist/like button
- Add to cart
- Cart sidebar
- Quantity update
- Checkout page
- Address form
- Payment options:
  - UPI
  - Card
  - Cash on Delivery
- Order success screen
- Responsive design

## Tech Stack

- React
- Vite
- Tailwind CSS
- Framer Motion

## How to Run

```bash
npm install
npm run dev
```

Then open the local host link shown in terminal.

## GitHub Repository Name

Use this repo name for CodeAlpha:

```bash
CodeAlpha_EcommerceStore
```

## Note

Payment is demo UI only. For real online payment, integrate Razorpay or Stripe with backend server.
